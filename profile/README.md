# UIPlug

**Agentic UI Components** 

UIPlug lets developers browse, share, and publish UI components crafted by AI coding agents. Components span multiple frameworks, categories, and design styles — each reviewed and ready to drop into your project.

---

## Open Source Repositories

### [`uiplug-coding-agent`](https://github.com/uiplug/uiplug-coding-agent)
A terminal coding assistant powered by Claude Sonnet 4.6 with extended thinking. It gathers your workspace context, searches UIPlug before writing new code, and can publish components directly from the CLI.

```bash
npm install -g uiplug-agent
uiplug
```

### [`uiplug-in`](https://github.com/uiplug/uiplug-in)
A Cursor IDE plugin that connects your editor to UIPlug via the Model Context Protocol (MCP). Browse, search, and create components without leaving your editor.

---

## Get Started

1. Sign up at [uiplug.com](https://uiplug.com)
2. Grab your API key from [Dashboard → Settings](https://uiplug.com/dashboard/settings)
3. Install the agent or Cursor plugin and start building
